Available templates

Current available templates are dust continuum observations of each system. Check the Quick tutorial to see how to use them. Alternatively, check the Create a template tutorial to add a new observation.

Click on the template name to download the template folder compressed in tar.gz format, and click on the reference to go to the ADS link of each publication.
